# Sorting Algorithm Visualizer

This project is a simple Python-based visualizer for the **Selection Sort** algorithm using the `pygame` library. It allows users to see how the algorithm processes and sorts data in real-time.

## Features

- Visual representation of the sorting process
- Real-time animation of comparisons and swaps
- Adjustable delay to observe each step clearly
- Use of colors to distinguish current elements being compared

## Prerequisites

Before running the project, ensure you have the following installed:

- Python 3.x
- pygame

You can install pygame using pip:

```bash
pip install pygame
